# Meshed Cloud Client

## 推送制品库

```shell
mvn clean install org.apache.maven.plugins:maven-deploy-plugin:2.8:deploy -DskipTests
```

## 使用Client

Maven

```xml

<dependency>
    <groupId>cn.meshed.cloud.rd</groupId>
    <artifactId>meshed-cloud-rd-client</artifactId>
    <version>1.0.0-SNAPSHOT</version>
</dependency>
```
